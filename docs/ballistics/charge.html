 15    """class that represent individual charges as standalone designs.
 16
 17    Parameters
 18    ----------
 19    density: float
 20        bulk density of the propellant, in kg/m^3. Reported value is close to
 21        1600 kg/m^3 for various modern, smokeless/nitrocellulose based propellant.
 22    force: float
 23        propellant force, or the work done by a kilogram of propellant gas (as
 24        ideal gas), when expanding from its isochoric adiabatic flame temperature
 25        to absolute zero, in an isentropic manner. Unit in J/kg, or (m/s)^2.
 26    pressure_exponent: float
 27        see the Notes section.
 28    covolume: float
 29        the co-volume of a propellant, as used in the Nobel-Abel equation of state:
 30        ```
 31        P (v-alpha) = RT
 32        ```
 33        where:
 34        - P: average pressure in Pa
 35        - v: specific volume of propellant gas, in m^3/kg
 36        - alpha: covolume, in m^3/kg
 37        - R: specific gas constant, in J/(kg-K)
 38        - T: average temprature in K
 39    adiabatic_index: float
 40        the (average) heat capacity ratio of the working gas while in-bore.
 41        At elevated temperatures and with a mix of species, this parameter typically
 42        clusters around 1.23-1.25.
 43    reduced_burnrate: float
 44        the burn-rate coefficient is factored against the shortest path to "burn through",
 45        or propellant's arch, to produce the `reduced burn rate`, defined as:
 46        ```
 47        u / e
 48        ```
 49        where:
 50        - u: burn rate coefficient, in m/s-Pa^n.
 51          - n: burn rate exponent, dimensionless.
 52        - 2e: width/thickness of the propellant arch.
 53    gas_molar_mass: float
 54        value used to calculate the average adiabatic index of a gas mixture.
 55        No particular unit is required, the only requirement being consistency across
 56        a set of `Charge` objects added to the same `ballistics.gun.Gun`. For the case
 57        of a single `Charge`, any value will work.
 58    arch_thickness: float
 59        twice the propellant's "web", or the minimum depth the propellant's burn surface
 60        must recede to complete combustion.
 61        see documentation of `ballistics.form_function` for more information.
 62
 63    Attributes
 64    ----------
 65    Z_k: float
 66        cached value from `ballistics.form_function.Z_k`.
 67
 68    Notes
 69    -----
 70    the Saint Robert's (Viellie's) burn rate law:
 71    ```
 72    u = a * P^n
 73    ```
 74    where:
 75    - u: linear burn rate, in m/s
 76    - a: burn rate coefficient, in m s^-1 Pa^-n
 77    - P: average chamber pressure, in Pa
 78    - n: pressure exponent, dimensionless.
 79    is used to model the combustion behavior of the propellant.
 80
 81    References
 82    ----------
 83    - **[English]** Xu, Fu-ming. (2013). On The Definition of Propellant Force.
 84    Defence Technology. 9. 127-130. 10.1016/j.dt.2013.10.005.

210    def areal_impulse(
211        self,
212        n_intg: int,
213        acc: float,
214        load_density: float = DEFAULT_LOAD_DENSITY,
215        ignition_pressure: float = DEFAULT_IGNITION_PRESSURE,
216        to: Significance = Significance.FRACTURE,
217    ) -> float:
218        """calculate the impulse-per-area of this propellant, as:
219        ```
220              t
221        I =   ∫ P(t) dt
222              0
223        ```
224        until the specified endpoint.
225
226        Parameters
227        ----------
228        n_intg: int
229            minimum number of steps taken from ignition to powder burnout.
230
231        acc: float
232            accuracy to which the ignition point is solved
233
234        load_density: float
235            the density to which the test bomb is loaded with propellant. The
236            default value of 0.2 g/cc (200 kg/m^3) is a common choice of loading
237            density for characterizing propellant performance.
238
239        ignition_pressure: float
240            the pressure developed by the ignition charge, usually fine-grained
241            black powder, which is taken as having fully burnt and inert during the
242            characterization of propellant. The default value is the typical ignition
243            pressure (of 100 kgf/sqcm) ascribed to by Chinese standardized testing
244            procedures.
245
246        to: `ballistics.Significance`
247            specifies to whence the p-t curve is integrated. Accepts either
248            `ballistics.Significance.BURNOUT` or `ballistics.Significance.FRACTURE`.
249            This concerns propellants of the multiple-perforated type, for which period
250            testing methodology only considers the combustion up to the fracture point.
251
252        Notes
253        -----
254        In propellant development, the characterization of propellant performance is
255        typically accomplished by a "closed-bomb" test, in which a sturdy pressure vessel
256        is loaded to a certain weight-of-charge-per-volume, known as the "load density",
257        then a small amount of ignition charge is used to initiate its combustion.
258        Propellant performance parameters can then be derived via data reduction of the
259        measured pressure-time curve.
260        The default values reflects Chinese standardized testing methodology as of 1970s
261        and 1980s.
262
263        References
264        ----------
265        - **[中文]** 第五机械工业部第二〇四研究所，火炸药手册（增订本）第三分册 火炸药分析
266        和测试，第五章第一节（1981.6)
267
268        Returns
269        -------
270        I: float
271            propellant's impulse-per-area up to the specified point. In unit of
272            kg/m-s
273
274        """
275        if to == Significance.BURNOUT:
276            Z_to = self.Z_k
277        elif to == Significance.FRACTURE:
278            Z_to = 1
279        else:
280            raise ValueError(
281                "`to` must be one of `Significance.BURNOUT` or `Significance.FRACTURE`"
282            )
283
284        """integrate backwards in time from burnout point to avoid asymptotic
285        point at t = 0, then read-off the time and impulse-values as negatives.
286        """
287        n, delta_t, rough_ttb = 0, MAX_DT, 0.0
288
289        while n < n_intg:
290            if rough_ttb > 0:
291                delta_t = rough_ttb / n_intg
292            n = 1
293
294            bs_next = BombState(
295                charge=self,
296                load_density=load_density,
297                ignition_pressure=ignition_pressure,
298                time=0,
299                burnup_fraction=0,
300                areal_impulse=0,
301            )
302
303            while bs_next.burnup_fraction < Z_to:
304                bs_now = bs_next
305                bs_next = self.propagate_rk4(bomb_state=bs_now, dt=delta_t)
306                n += 1
307
308            rough_ttb = bs_next.time
309
310        tol_t = rough_ttb * acc
311
312        def time_burnup(time: float) -> float:
313            return (
314                self.propagate_rk4(
315                    bomb_state=bs_now, dt=time - bs_now.time
316                ).burnup_fraction
317                - Z_to
318            )
319
320        time_to = dekker(f=time_burnup, x_0=bs_now.time, x_1=bs_next.time, tol=tol_t)[0]
321        bs_to = self.propagate_rk4(bomb_state=bs_now, dt=time_to - bs_now.time)
322
323        return bs_to.areal_impulse
324
325    def dt(self, bomb_state: BombState) -> BombDelta:
326        P = bomb_state.pressure
327        return BombDelta(d_time=1, d_burnup_fraction=self.dZdt(P), d_areal_impulse=P)
328
329    def propagate_rk4(self, bomb_state: BombState, dt: float) -> BombState:
330        s_i = bomb_state.increment
331        df = self.dt
332
333        k1 = df(bomb_state)
334        k2 = df(s_i(d=0.5 * k1 * dt, dt=0.5 * dt))
335        k3 = df(s_i(d=0.5 * k2 * dt, dt=0.5 * dt))
336        k4 = df(s_i(d=k3 * dt, dt=dt))
337        return s_i(d=(k1 + k2 * 2 + k3 * 2 + k4) * dt / 6, dt=dt)

330    def propagate_rk4(self, bomb_state: BombState, dt: float) -> BombState:
331        s_i = bomb_state.increment
332        df = self.dt
333
334        k1 = df(bomb_state)

class that represent individual charges as standalone designs.

Parameters
  • density (float): bulk density of the propellant, in kg/m^3. Reported value is close to 1600 kg/m^3 for various modern, smokeless/nitrocellulose based propellant.
  • force (float): propellant force, or the work done by a kilogram of propellant gas (as ideal gas), when expanding from its isochoric adiabatic flame temperature to absolute zero, in an isentropic manner. Unit in J/kg, or (m/s)^2.
  • pressure_exponent (float): see the Notes section.

  • covolume (float): the co-volume of a propellant, as used in the Nobel-Abel equation of state:

    P (v-alpha) = RT

    where:

    • P: average pressure in Pa
    • v: specific volume of propellant gas, in m^3/kg
    • alpha: covolume, in m^3/kg
    • R: specific gas constant, in J/(kg-K)
    • T: average temprature in K
  • adiabatic_index (float): the (average) heat capacity ratio of the working gas while in-bore. At elevated temperatures and with a mix of species, this parameter typically clusters around 1.23-1.25.

  • reduced_burnrate (float): the burn-rate coefficient is factored against the shortest path to "burn through", or propellant's arch, to produce the reduced burn rate, defined as:

    u / e

    where:

    • u: burn rate coefficient, in m/s-Pa^n.
      • n: burn rate exponent, dimensionless.
    • 2e: width/thickness of the propellant arch.
  • gas_molar_mass (float): value used to calculate the average adiabatic index of a gas mixture. No particular unit is required, the only requirement being consistency across a set of Charge objects added to the same ballistics.gun.Gun. For the case of a single Charge, any value will work.
  • arch_thickness (float): twice the propellant's "web", or the minimum depth the propellant's burn surface must recede to complete combustion. see documentation of ballistics.form_function for more information.
Attributes
  • Z_k (float): cached value from ballistics.form_function.Z_k.
Notes

the Saint Robert's (Viellie's) burn rate law:

u = a * P^n

where:

  • u: linear burn rate, in m/s
  • a: burn rate coefficient, in m s^-1 Pa^-n
  • P: average chamber pressure, in Pa
  • n: pressure exponent, dimensionless. is used to model the combustion behavior of the propellant.
